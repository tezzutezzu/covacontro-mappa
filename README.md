# mappa-covacontro

Simple project containing a single static `index.html` page.

## Quick start

- Open [index.html](index.html#L1) in your browser.
- Or serve the folder with a simple static server, for example:

```bash
python3 -m http.server 8000
# then open http://localhost:8000/
```

## Notes

- Created for local use. No build step required.
